# terminal-verdict
10cpt: end of year task - group python game jam project

Prototype scaffold added for `TERMINAL.EXIT` game.

Run the prototype locally:

1. (optional) Create a virtualenv and install deps:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

2. Run the game:

```bash
python3 main.py
```

This creates a minimal, easy-to-extend codebase under `terminal_exit/`.
